# Projeto: Banco de Dados E-commerce (Modelo Refinado)

## Descrição

Este projeto implementa o modelo lógico de um sistema de e-commerce com:

- Cliente PF ou PJ (exclusivo)
- Múltiplas formas de pagamento por cliente
- Entrega com status e código de rastreio

Normalizado até a 3FN.

## Estrutura

```
ecommerce-db
 ┣ schema.sql
 ┣ inserts.sql
 ┣ queries.sql
 ┗ README.md
```

## Como usar

1. Execute o schema.sql
2. Execute o inserts.sql
3. Teste com queries.sql
